# Overview

A Work-In-Progress networked twin-stick shooter based on Battlezone (1998) and Tanks! (Wii Play; 2006), made in the Unity engine. The prototype has 8 playable levels and 14 playable vehicles, each featuring a unique weapon, which can all be played in multiplayer via efficient custom P2P networking. Vehicle models were custom made, weapon effects were sourced from the Unity asset store, and sounds were created using assets from freesound.org.

# Features

- Multiple playable levels, vehicles, and weapons
- Efficient P2P networking to support the fast-paced action
- Accolade system to reward emergent milestones (e.g. Killing Spree after defeating 5 enemies without dying)
- Genericised & modular enemy AI to support a range of vehicles and weapons
- Complex upgrade system including augmentations and weapon enhancements
